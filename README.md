## 1. 系统架构与运行机制

* **部署环境**：完全托管于 GitHub Actions 云端，本地无需开机。
* **触发机制**：Cron 定时任务。**每周六北京时间上午 10:00** 自动运行并推送 Server酱 微信通道。
* **时空锁定算法**：引擎内置“日线降维与周五强制锚定”机制。无论周末何时触发运行，系统会自动切除本周末的未闭合数据幻影，强行对齐到**最近一个日历周五的 23:59:59**，确保与所有正规行情软件的周线绝对同频。
* **多源异构数据引擎**：
  * **股票/场内 ETF (`yf`)**：主战线使用 Yahoo Finance，无缝搭载东方财富 (Eastmoney) 与腾讯财经双模容错底层，自动绕过云端 IP 封锁，智能处理美股代码变异。
  * **场外公募基金 (`jj`)**：直连东方财富天天基金网，提取精准单位净值。
  * **加密货币 (`crypto`)**：彻底抛弃传统爬虫，直连美国最大合规交易所 **Coinbase Pro** 的机构级公开 API，实现 7x24 小时抗封锁极速穿透。

## 2. 核心配置库：`portfolio.csv` 维护指南 (⚠️最重要)

系统的所有监控目标及展示流向，全部由仓库根目录下的 `portfolio.csv` 控制。
**如需增减持仓、修改名称、调整监控池，只需在 GitHub 网页端点击 `portfolio.csv` -> 点击右上角“铅笔”图标编辑 -> 提交保存即可。**
